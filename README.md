## What This Is

Every paper in this repository was produced by an autonomous, milestone-gated
paper pipeline implementing the five-component DVL framework:

1. **Immutable Specification** — frozen parameters locked before writing begins
2. **Prior-Drift Detection** — adversarial review validates against the frozen spec
3. **Adversarial Role Separation** — Author, Peer Reviewer, and Editor operate under distinct personas with opposing incentives
4. **Statistical Validation** — sensitivity analysis, boundary conditions, and competing model rejection required in every paper
5. **Persistent State Management** — full audit trail, innovation log, and reproducibility artifacts for every run

Each paper includes formal proofs (Lean-ready structure), adversarial boundary
conditions, sensitivity analysis, and verified citations.
